About Detector

Body Detection

You can start using body detection by the webcam. To detect, you just need to choose any part of the body that you want. No matter what you do, your virtual character will follow you to do the same action.

Action Recording

Before recording, select the recording distance first. You can record only when the distance between you and the camera matches the distance you selected. When you start recording, the system will record the rotation angle of your body parts. After recording, you can export JSON files that record actions.

Theme and Layouts

During detection, you can choose your favorite theme color and layout. Our system has a variety of beautiful themes for you to choose from. You can also adjust the layout size according to your needs.

Many body parts are detectable

All parts of the human body can be detected. The part of the hand can be divided into arms, wrists and fingers. In addition, the part of the eye can also detect not only the blink, but also the direction of sight, and even the change of the mouth shape can be detected.

About Editor

You can import your pre-recorded JSON file and play. During the process, you can click the buttons of pause, stop, forward five seconds, backward five seconds and loop. You can also record or upload audio files and edit them. Even you can mark some actions, and then the system will pause for three seconds when the marked action is being played. In the future, when the user performs this action, it can detect whether the user has performed the corresponding action.

WARNING

Please pay attention to the following rules.

Actions to Be Alerted

★ Please play the JSON file before editing.

★ Please choose the file before pressing the play button.

★ Please notice not to import the same file.

★ Please make sure to export the file that you had edited before deleting the file or leaving the website.

★ Please do not mark the same body part at the same time in the file.

★ If the JSON file that has recorded sound is marked again, the length of the sound and the animation will be different, so if you want to mark it, you may need to record the sound again.

★ Because the recorded sound is used to be played by the users, so the length of the sound will include the paused time of all marked time. Therefore, before you play the JSON file, please remember to press the bookmark button to synchronize the sound with the animation.


The Rules of Playing

◎ You cannot press the pause button when no file is playing.

◎ When the bookmark button is pressed, the animation will pause for three seconds if the time is marked, and the pause, forward five seconds and backward five seconds buttons cannot be clicked during the pause.

The Rules of Editing

◎ The left and right sides of the body parts in the dropdown menu correspond to the sides of the user.

◎ You can name this marked action. If you don't name it, it will be presented as untitled.

◎ When you edited your file, the export light will turn red, please remember to export your file to save it.


The Rules of Recording and Playing Sound

◎ During the recording process, you can decide whether to mute or not at any time, and you can also record the audio file multiple times.

◎ You can press the recording pause button at any time to pause the animation and recording. However, when the animation is played to the time that was marked, you can not press the recording pause button.

◎ You can't stop recording after you start recording sound

◎ During recording, the buttons for play, pause, stop, forward five seconds, backward five seconds, and loop cannot be clicked, and the bookmark button will be automatically checked and then become unclickable.

◎ When playing a JSON file, the selected audio file will be played. However, when exporting JSON file, all audio files will be exported instead of only the selected audio file. You can also click any sound file you want to export.

About Player

Import the edited JSON file, and users can play the edited file through the player. Users can import multiple files at the same time, and then select the animation that you want to watch. When the player plays to the time which was marked, it will count down three seconds and show the action name. Before the countdown is over, you need to follow the virtual character to do the corresponding actions, and then you will be rated according to the accuracy of your actions.

The Rules of Player

Distance detection

Before using the player, you need to detect whether the distance between you and the camera matches the distance set in the JSON file.

About Pause Button

When the animation is played to the time which was marked or detecting the distance between you and the camera, you can not press the pause button.

Direction of the actions

During the process, the user needs to make mirrored actions according to the virtual character.

Export Files

The video and action results of the user will be recorded during the process. At the end, the video file and the PDF file of the results can be exported.
